## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Stack

* Nest
* MongoDB

## Installation

1. Install dependencies
```bash
$ yarn install
```

2. Intall Nest CLI globally
```bash
$ npm i -g @nestjs/cli
```

3. Start database
```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
